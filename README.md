# ⚡ CTRX SUBDOMAIN

> **Fast and Powerful Subdomain Management Script — Beta Test**

CTRX SUBDOMAIN is a lightweight yet powerful PHP-based solution for managing subdomains easily from a web interface. Whether you're launching microsites, managing user spaces, or automating domain deployment, CTRX SUBDOMAIN gives you full control — all from your browser.

---

## 🚀 Features

### 🖥️ Frontend (User Panel)
- 🌐 **Landing Page**
- 🧑‍💻 **Create Subdomain**
- ❌ **Delete Subdomain**
- 📃 **View Subdomain List**

### 🛠️ Backend (Admin Panel)
- ✏️ **Edit Subdomain**
- ❌ **Delete Subdomain**
- 🔐 **View License Details**
- ➕ **Add Domain**
- 🗑️ **Delete Domain**
